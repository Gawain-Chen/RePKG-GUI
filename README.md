# RePKG-WPF

#### 用于解压PKG文件的GUI图形界面

## 项目介绍：

1.可以解压后辍名称为.pkg的文件

2.支持多选文件，累计添加

3.解压程序后台运行，不影响前台流畅

## 使用方法：

1.打开程序

2.选择一个以上的pkg文件

3.如果需要，可以选择输出目录，默认目录为桌面文件夹

4.单击“开始”即可开始解压pkg文件

5.可以单击“打开输出目录”来查看解压后的文件
